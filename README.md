# PleiadianOS is an Open-Source NixOS based distribution designed to have Windows 11 functionality while combining macOS GUI for ease of use in order to create a Linux experience with enough power and stability for the Linux newcomer to complete their daily tasks while transistioning to our Open-Source Community.
